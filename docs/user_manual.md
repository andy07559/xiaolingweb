## 插件简介

小灵内容提取是一款强大的Chrome浏览器插件，专门用于网页内容的智能提取、分析和处理。它能够自动识别网页主要内容，提供多种格式的导出选项，支持AI智能分析和多语言翻译功能。

### 核心特点
- 智能内容识别与提取
- 多格式内容导出
- AI智能分析
- 多语言翻译支持
- 图片和表格提取
- 文本翻译工具
- 一键处理功能（新）
- 自定义URL支持（新）

## 主要功能

### 1. 内容提取
- 一键处理（提取+分析+翻译）
- 自定义URL内容提取
- 自动识别网页主要内容
- 支持纯文本、HTML、Markdown三种格式
- 智能过滤广告和无关内容
- 保持原文格式和结构

### 2. 内容分析
- 自动识别文章主题
- 提取关键词
- 统计字数
- 计算预计阅读时间

### 3. AI总结
- 支持多种总结风格
  - 默认总结
  - 简短精炼
  - 详细分析
  - 创意解读
  - 学术分析
- 可自定义提示词

### 4. 翻译功能
- 支持多种语言互译
- 多种翻译风格选择
- 独立文本翻译工具
- 快速复制翻译结果

### 5. 媒体提取
- 提取网页图片
- 提取表格数据
- 显示图片详细信息
- 保持表格原始格式

## 使用指南

### 1. 内容提取

方式一：一键处理（新增功能）
1. 在插件面板中输入要提取的网页URL（可选，默认使用当前页面）
2. 点击"一键提取+分析+翻译"按钮
3. 系统将自动执行：
   - 提取网页内容
   - 生成AI详细分析总结
   - 自动检测并翻译非中文内容
4. 等待处理完成后查看结果：
   - 提取的内容（如果原文不是中文则显示翻译后的内容）
   - AI总结结果
   - 内容分析结果

方式二：通过插件图标
1. 点击Chrome工具栏中的插件图标
2. 输入要提取的网页URL（可选，默认使用当前页面）
3. 选择输出格式（纯文本/HTML/Markdown）
4. 点击"提取内容"按钮

### 2. 查看分析结果
- 提取完成后自动显示分析结果
- 包含主题、关键词、字数等信息
- 可查看预计阅读时间

### 3. AI总结功能

1. 提取内容后，点击"AI总结"按钮
2. 等待几秒钟，AI生成的总结会显示在内容下方
3. 可以：
   - 点击"复制总结"将内容复制到剪贴板
   - 点击"保存总结"将内容保存为文本文件

注：使用一键处理功能时，AI总结会自动生成，无需单独点击。

### 4. 翻译内容
- 点击"翻译内容"按钮
- 选择目标语言
- 选择翻译风格（如适用）
- 等待翻译完成
- 可复制翻译结果

### 5. 查看媒体内容
- 点击"查看图片/表格"按钮
- 在弹出窗口中浏览图片
- 查看表格数据
- 可关闭窗口继续其他操作

### 文本翻译工具使用
1. 在文本框中输入或粘贴要翻译的文本
2. 点击"翻译"按钮
3. 等待翻译完成
4. 查看翻译结果

## 功能详解

### 输出格式说明

1. **纯文本格式**
   - 仅保留文字内容
   - 清除所有格式
   - 适合快速复制使用

2. **HTML格式**
   - 保留原始网页格式
   - 包含样式信息
   - 适合需要保持排版的场景

3. **Markdown格式**
   - 转换为Markdown语法
   - 保留基础格式
   - 适合写作和编辑

### AI总结风格说明

1. **默认总结**
   - 平衡详细度和简洁性
   - 突出重点内容
   - 适合一般使用场景

2. **简短精炼**
   - 200字以内的简要总结
   - 只包含核心信息
   - 适合快速了解

3. **详细分析**
   - 全面的内容分析
   - 包含多个维度
   - 适合深入研究

4. **创意解读**
   - 独特的视角
   - 联系实际应用
   - 适合创意工作

5. **学术分析**
   - 学术角度的分析
   - 包含方法论评估
   - 适合学术研究

### 翻译功能说明

1. **支持的语言**
   - 中文（默认目标语言）
   - 英语
   - 日语
   - 韩语
   - 法语
   - 德语
   - 西班牙语
   - 俄语
   - 更多语言支持

2. **翻译风格**
   - 正式风格（商务/学术）
   - 日常风格（非正式交流）
   - 专业风格（技术文档）
   - 创意风格（文学创作）

## 常见问题

### 1. 内容提取失败
- 检查网页是否完全加载
- 刷新页面后重试
- 确认网页内容是否可访问
- 如果使用自定义URL，确保URL格式正确且可访问

### 2. 翻译功能无响应
- 检查网络连接
- 确认API配置是否正确
- 刷新页面重试

### 3. AI总结失败
- 检查API Key设置
- 确认内容长度是否在限制范围内
- 检查网络连接

### 4. 媒体提取问题
- 确认页面包含图片/表格
- 检查图片是否可访问
- 刷新页面后重试

### 5. 一键处理相关问题
- 如果处理中断，可以尝试分步骤处理
- 确保网络稳定，因为需要连续调用多个API
- 如果内容较多，处理时间可能较长，请耐心等待
- 如果翻译步骤失败，但其他步骤成功，可以手动使用翻译功能

### 6. 自定义URL问题
- 确保输入的URL完整（包含http://或https://）
- 某些网站可能有访问限制，建议使用公开可访问的网页
- 如果URL无效，系统会自动使用当前页面
- 跨域限制可能导致某些网站无法提取，建议打开目标网页后再提取

## 注意事项

1. **使用限制**
   - 部分功能需要配置API Key
   - 某些网站可能限制内容提取
   - 处理大量内容时可能需要等待

2. **隐私安全**
   - 不会收集用户个人信息
   - 所有数据本地处理
   - API调用遵循安全协议

3. **最佳实践**
   - 定期清理缓存
   - 及时更新插件版本
   - 保存重要内容到本地
